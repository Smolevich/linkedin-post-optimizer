# Examples

## Draft Example

Raw idea input:

```
I use several AI coding tools every day. Want to write about my setup:

- Claude Code (CLI) - $20/month, use it 70% of time
- Codex CLI - $30/month, great for code review
- Antigravity (IDE) - free tier is enough

Total: $50/month

I constantly hit rate limits on all of them. Have to switch between tools.

Key insight: at developer rates, this pays for itself in 1 hour of work.

Question for audience: what tools do you use and what's your budget?

Notes:
- This is for RU audience
- Want to spark discussion about AI tool costs
- Should include ROI angle
```

## Output Example — Variant 1 (Personal Stats)

**Hook:**
$50/мес на AI-агенты. Исчерпываю даже недельные квоты.

**Body:**
Сейчас компании выпускают инструменты в двух форматах: CLI и IDE. Моя статистика: 70% времени я в CLI, 30% — в IDE.

Я выделил для себя 3 инструмента:

**Claude Code (CLI)**
Плачу $20/мес (Pro). Окупается мгновенно — агент сам ходит в bash и читает файлы.
- Skills: Самая мощная вещь.
- MCP: Chrome extension стабильнее playwright.

**Codex (CLI)**
Плачу $30/мес (Business). Мой идеальный помощник.
- Ревью: Использую в связке с Claude Code.
- Разработка: Пишу код и создаю приложения.

**Antigravity (IDE)**
Инструмент для 30% сложных задач.
- Стабильнее Cursor (от которого отказался).
- Хватает бесплатных лимитов на модели Google/Claude/OpenAI.

Итого: $50/мес. При ставке разработчика это окупается за 1 час работы (и экономит минимум 10 часов рутины в неделю).

**CTA:**
А что вам ближе: CLI-first или IDE-first? И какой месячный бюджет закладываете на инструменты?

**Word count:** ~195 words
